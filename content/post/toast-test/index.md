这篇文章用于在真实场景中测试 **Toast 消息提示组件**。

## 代码块复制测试

点击下方代码块右上角的复制按钮，应弹出「代码已复制」的成功提示：

```python
def greet(name: str) -> str:
    """一个简单的问候函数"""
    return f"Hello, {name}! 欢迎使用 Toast 组件。"

if __name__ == "__main__":
    print(greet("Kittychen"))
```

## 底部按钮测试

文章正文下方有三个按钮，分别测试三种提示类型：

- **🔗 复制本文链接** —— 真实复制当前 URL 到剪贴板，成功后弹出绿色成功提示
- **⏰ 稍后阅读** —— 演示橙色警告提示
- **⚠ 模拟操作失败** —— 演示红色错误提示（含较长文本）

所有提示均为 3 秒后自动消失，也可点击右上角 × 或遮罩手动关闭。
